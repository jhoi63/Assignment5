WEB700 Course at Seneca College

The WEB700 course at Seneca College is a comprehensive program covering various aspects of web development.

Course Themes by Week

  1. Week 1: Course Introduction and Internet Architecture - This week provides an overview of the course, internet architecture, and the development environment. Students get introduced to JavaScript basics.
  2. Week 2: JavaScript Fundamentals - This week covers more advanced JavaScript concepts such as functions, scope, closures, and built-in objects.
  3. Week 3: Object-Oriented JavaScript and ES6 Features - Students delve into object-oriented programming in JavaScript and explore ES6 features like classes and arrow functions.
  4. Week 4: Introduction to Node.js - Node.js fundamentals are covered, including modules, npm, and building web servers with Express.js.
  5. Week 5: Express.js Introduction and Deployment - Further exploration of Express.js, focusing on routing, middleware, and deployment considerations.
  6. Week 6: HTML5 Introduction - Basics of HTML5, document structure, common elements, and semantic HTML are covered.
  7. Week 7: CSS Introduction - This week covers CSS fundamentals like selectors, colors, box model, and positioning.
  8. Week 8: Forms Introduction and Processing Forms with Express.js - Students learn about HTML forms and processing them using Express.js.
  9. Week 9: Template Engines - Handlebars.js - Introduction to template engines in Express.js, focusing on Handlebars.js for server-side rendering.
  10. Week 10: Introduction to AJAX and REST API - Students learn about AJAX for asynchronous data retrieval and RESTful APIs.
  11. Week 11: Introduction to Postgres and Sequelize ORM - Basics of setting up a Postgres database and using Sequelize ORM for data manipulation.
  12. Week 12: Security Considerations - The course concludes with a focus on security in web development, covering HTTPS, password encryption, and other security best practices.
