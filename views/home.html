Objective

Build upon the code created in Assignment 3 by incorporating the Handlebars view engine to render JSON data visually in the browser using .hbs views and layouts. Additionally, update the officeData and server.js modules to allow for viewing and adding employees individually using a web form.

Specification

Part 1: Getting Express Handlebars & Updating Views

  • Install and configure the "express-handlebars" module
  • Create a "layouts" folder inside the "views" folder
  • Create a "main.hbs" file inside the "layouts" folder

Part 2: Updating the Employees Route & Adding a View

  • Create an "employees.hbs" file inside the "views" folder
  • Update the GET "/employees" route to render employee data using res.render()
  • Build the employees table in the "employees.hbs" file

Part 3: Adding a View and Route to Support Adding Employees

  • Create an "addEmployee.html" file inside the "views" folder
  • Add the GET "/employees/add" route in server.js
  • Add the POST "/employees/add" route in server.js
  • Create the "addEmployee" function in officeData.js

Part 4: Adding Description View & Updating server.js

  • Create a "description.hbs" file inside the "views" folder
  • Add content to explain the code in "main.hbs" and the input elements in "addEmployee.html"
  • Update the GET "/description" route in server.js

Part 5: Publishing your site (optional)

Publish your site online once it has been tested locally and verified.
